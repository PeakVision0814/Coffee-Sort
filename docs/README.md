## 🚀 推荐阅读顺序

为了让你更顺畅地跑通项目，建议按照以下顺序阅读：

1.  **环境准备**：进入 [Environment Setup](./Environment%20Setup/README.md) 完成 Python 环境和驱动的安装。
2.  **硬件连接**：查阅 [硬件连接自查](./myCobot_280_M5/2.%20基础设置/4.%20首次安装/4.1%20首次使用自查.md)，确保机械臂与电脑连接正常。
3.  **开始实战**：环境和硬件就绪后，返回项目根目录阅读主 `README.md`，运行主程序开始分拣！
